# Phase 1 Vim Basics

## 目的

- Vim の基本操作で困らない状態にする
- 編集、保存、検索、置換、undo / redo を行えるようにする
- 今後の Linux 学習で最低限必要な Vim 操作を身につける

## 学習項目

- 保存 / 終了
- モード理解
- 移動
- 編集
- 検索
- 置換
- undo / redo
- 最低限の効率操作

## 基本操作まとめ

### モード

- ノーマルモード
  - 移動や削除、コピー、貼り付けなどの操作を行うモード
- インサートモード
  - 文字入力を行うモード
- コマンドラインモード
  - `:` から始まる保存、終了、置換などのコマンドを入力するモード

## 保存 / 終了

- `:w`
  - 保存
- `:q`
  - 終了
- `:wq`
  - 保存して終了
- `:q!`
  - 保存せず終了

## 移動

- `h`
  - 左へ
- `j`
  - 下へ
- `k`
  - 上へ
- `l`
  - 右へ
- `w`
  - 次の単語の先頭へ
- `b`
  - 前の単語の先頭へ
- `0`
  - 行頭へ
- `$`
  - 行末へ
- `gg`
  - ファイル先頭へ
- `G`
  - ファイル末尾へ
- `10G`
  - 10 行目へ

## 編集

- `i`
  - 現在位置の前から挿入
- `a`
  - 現在位置の後ろから挿入
- `o`
  - 次の行を開いて挿入
- `x`
  - 1 文字削除
- `dd`
  - 1 行削除
- `yy`
  - 1 行コピー
- `p`
  - 貼り付け

## 検索 / 置換

- `/word`
  - `word` を前方検索
- `n`
  - 次の検索結果へ
- `N`
  - 前の検索結果へ
- `:%s/old/new/g`
  - ファイル全体で `old` を `new` に置換

## 取り消し / やり直し

- `u`
  - undo
- `Ctrl r`
  - redo

## 効率操作

- `ciw`
  - カーソル位置の単語を変更
- `diw`
  - カーソル位置の単語を削除
- `dd`
  - 行削除
- `yy`
  - 行コピー
- `p`
  - 貼り付け

## 実際にやったこと

### `vimtutor`

最初に `vimtutor` を実施した。

```bash
vimtutor
```

ここで確認したこと。

- Vim はモードを意識するエディタである
- ノーマルモードとインサートモードの切り替えに慣れる必要がある

### Kata 001: 基本編集

対象ファイル:

```bash
katas/01_vim/001_basic-edit/sample.txt
```

修正前:

```bash
Linxu is fun.

```

修正後:

```bash
Linux is fun.

I want to learn command line.
```

この演習で使った主な操作。

- `w`
- `b`
- `i`
- `a`
- `x`
- `:wq`

### Kata 002: 検索 / 置換

対象ファイル:

```bash
katas/01_vim/002_search-replace/log.txt
```

実施内容。

- `error` を検索
- `n`, `N` で検索結果を移動
- 全体置換を実施

使用した置換コマンド:

```bash
:%s/error/ERROR/g
```

### Kata 003: 移動 / undo / redo

対象ファイル:

```bash
katas/01_vim/003_navigation/notes.txt
```

実施内容。

- `gg` で先頭へ移動
- `G` で末尾へ移動
- `10G` で 10 行目へ移動
- `0`, `$` で行頭 / 行末へ移動
- `dd` で 10 行目削除
- `u` で undo
- `Ctrl-r` で redo
- 最後は undo で元に戻して保存
