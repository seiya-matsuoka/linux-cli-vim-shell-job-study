# Phase 2 Text Katas

## 目的

Phase 2 のテキスト処理・ファイル操作の実戦化で扱う練習問題をまとめたディレクトリ。

このディレクトリでは、以下の基礎を実際に手を動かして確認する。

- `grep`
- `ripgrep`
- `find`
- `xargs`
- `sort`
- `uniq`
- `wc`
- `cut`
- `tr`
- `diff`
- `sed`
- `awk`

## 進め方

各サブディレクトリの `README.md` を読み、課題に沿って操作を行う。

各課題では、必要に応じて以下も残す。

- `workspace/` 内の入力ファイルや演習用ファイル
- `artifacts/` 側に保存した実行結果
- `docs/` 側にまとめた学習メモ

## 課題一覧

- `001_grep-rg/`
  - `grep`, `ripgrep` の基本
- `002_find-xargs/`
  - `find`, `xargs` の基本
- `003_sort-uniq-wc-cut-tr/`
  - 並べ替え、重複整理、件数確認、列抽出、文字変換
- `004_diff/`
  - 差分確認
- `005_sed/`
  - 置換と抽出
- `006_awk/`
  - 列処理と集計
