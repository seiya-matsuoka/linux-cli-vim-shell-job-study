# Kata 004: Functions

## 目的

関数を使って処理を分割できるようにする。

## 学習対象

- 関数定義
- `local`
- コマンド置換

## 作業ディレクトリ

```bash
katas/03_shell/004_functions/workspace/
```

## 課題

以下を順に実行する。

1. `say_hello` 関数を作る
2. `sum_two_numbers` 関数を作る
3. 関数の出力をコマンド置換で受け取る
4. 実行結果を確認する

## 実施コマンド例

```bash
chmod +x functions.sh
./functions.sh
```

## 確認ポイント

- 関数で処理を分割できる
- 関数内変数は `local` を使う
- `echo` した値をコマンド置換で受け取れる
