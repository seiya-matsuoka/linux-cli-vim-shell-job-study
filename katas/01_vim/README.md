# Phase 1 Vim Katas

## 目的

Phase 1 の Vim 基礎学習で扱う練習問題をまとめたディレクトリ。

このディレクトリでは、以下の基本操作を反復する。

- 保存 / 終了
- モード理解
- 移動
- 編集
- 検索
- 置換
- undo / redo
- 最低限の効率操作

## 進め方

1. 最初に `vimtutor` を実施する
2. 各サブディレクトリの `README.md` に従って課題を行う
3. 編集結果やログを必要に応じて `artifacts/` に保存する

## 課題一覧

- `001_basic-edit/`
  - 基本的な編集と保存
- `002_search-replace/`
  - 検索と置換
- `003_navigation/`
  - 移動、undo、redo
