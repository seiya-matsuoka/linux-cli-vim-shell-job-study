# Phase 1 CLI Katas

## 目的

Phase 1 の CLI 基礎学習で扱う練習問題をまとめたディレクトリ。

このディレクトリでは、以下の基礎を実際に手を動かして確認する。

- ファイル / ディレクトリ操作
- パスの理解
- リダイレクト / パイプ
- exit code
- 引用符とエスケープ
- `man` / `--help` / `info`
- プロセス / ジョブ制御の入口

## 進め方

各サブディレクトリの `README.md` を読み、課題に沿って操作を行う。

各課題では、必要に応じて以下も残す。

- `workspace/` 内の入力ファイルや演習用ファイル
- `artifacts/` 側に保存した実行結果
- `docs/` 側にまとめた学習メモ

## 課題一覧

- `001_files-and-dirs/`
  - ファイル / ディレクトリ操作の基礎
- `002_paths/`
  - 絶対パス、相対パス、`~`、`.`、`..`
- `003_redirect-and-pipe/`
  - `>`, `>>`, `2>`, `2>&1`, `|`
- `004_exit-code/`
  - `$?`, `&&`, `||`
- `005_quotes-and-escape/`
  - シングルクォート、ダブルクォート、エスケープ
- `006_help-man-info/`
  - `man`, `--help`, `info`
- `007_process-and-job/`
  - `ps`, `top`, `&`, `jobs`, `fg`, `bg`, `kill`
